¿Qué necesitaréis para poder seguir el curso?

1. Rstudio & R (es necesario instalar el paquete stylo)
	https://www.rstudio.com
	https://www.r-project.org/about.html
	Dependiendo de vuestro sistema operativo es posible que tengáis que instalar programas de entorno. Aquí podéis encontrar las descripciones detalladas: https://cran.r-project.org/web/packages/nat/vignettes/Installation.html


2. Python (favor de instalar las siguientes librerías: seaborn, matplotlib, numpy, statsmodel, pandas, progress, fonemas, silabeador)
	https://www.python.org

Python ofrece un programa que ayuda en la instalación de librerías. Se llama "pip":
https://pypi.org/project/pip/

	Para instalar fonemas y silabeador podéis usar el pip o descargarlos de los siguientes repositorios github repositories: https://github.com/fsanzl/fonemas; https://github.com/fsanzl/silabeador)
